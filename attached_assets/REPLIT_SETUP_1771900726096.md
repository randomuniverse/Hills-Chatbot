# 🐾 펫 라이프 플래너 — Replit 세팅 가이드

## 1단계: 파일 구조 만들기

Replit에서 아래 구조로 파일을 만들어요:

```
hills-pet-planner/
├── index.html
├── package.json
├── vite.config.js
├── src/
│   ├── main.jsx
│   └── App.jsx
│   └── App.css
└── backend/
    ├── main.py
    └── requirements.txt
```

## 2단계: Secrets 등록 (매우 중요!)

Replit 왼쪽 메뉴 🔒 **Secrets** 탭에서 3개 등록:

| Key | 값 가져오는 곳 |
|-----|--------------|
| `SUPABASE_URL` | Supabase → Project Settings → API → Project URL |
| `SUPABASE_ANON_KEY` | 같은 화면 → anon public 키 복사 |
| `ANTHROPIC_API_KEY` | console.anthropic.com → API Keys |

## 3단계: .replit 파일 설정

Replit에 `.replit` 파일을 만들고 아래 내용 붙여넣기:

```toml
run = "bash start.sh"
```

## 4단계: start.sh 파일 만들기

```bash
#!/bin/bash

# 백엔드 의존성 설치 및 실행
pip install -r backend/requirements.txt -q
uvicorn backend.main:app --host 0.0.0.0 --port 8000 &

# 프론트엔드 의존성 설치 및 실행
npm install
npm run dev -- --host 0.0.0.0 --port 3000
```

## 5단계: 실행

Replit 상단 **Run** 버튼 클릭!

잠시 후 오른쪽 미리보기에 챗봇이 나타나요.

---

## 문제 해결

**백엔드 오류**: Secrets에 키가 제대로 등록됐는지 확인
**프론트 안 보임**: npm install 후 새로고침
**추천 안 됨**: Supabase products 테이블에 데이터가 있는지 확인
